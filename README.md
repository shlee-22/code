Decoding
1. The decoding pipeline is implemented in Python. (Python 3.12)
2. Channel-wise information can be provided in a dictionary format for execution.
3. To run the decoding pipeline, execute the Python scripts located in the Decoding directory.
4. User data can be analyzed by importing an input dataset in Microsoft Excel (.xlsx) format.

Analysis
1. Neural data analysis is implemented in MATLAB. (MATLAB R2022a)
2. It is used for analyzing channel-wise neural signals.
3. Data can be imported after being exported from TDT.
4. User data can be analyzed by importing an input dataset in Microsoft Excel (.xlsx) format.

ML 
1. The machine learning pipeline is implemented in Python. (Python 3.12)
2. It utilizes an SVM algorithm to compare movements between groups.
3. To run the machine learning pipeline, execute run_experiment.py in the ML directory.
4. User data can be analyzed by importing an input dataset in Microsoft Excel (.xlsx) format.
5. Please refer to the README file within the ML directory for more details.
